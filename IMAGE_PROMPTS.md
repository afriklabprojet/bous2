## Instructions d'utilisation :

1. **DALL-E (ChatGPT Plus)** :

   - Copiez le prompt
   - Collez dans ChatGPT
   - Téléchargez l'image générée

2. **Midjourney** (Discord):

   - `/imagine` suivi du prompt
   - Ajoutez `--ar 4:3 --v 6` à la fin
   - Upscale l'image que vous préférez

3. **Stable Diffusion** (en ligne):
   - Utilisez sur replicate.com ou clipdrop.co
   - Collez le prompt
   - Générez et téléchargez

## Après génération :

- Nommez les fichiers : `maple-wood.jpg`, `birch-wood.jpg`, etc.
- Placez-les dans `/public/images/`
- Dites-moi quand c'est fait, je mettrai à jour le code
